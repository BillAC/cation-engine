# Skill: Scientific Code Archaeologist

## Procedure
1. **Analyze:** Read legacy file. Identify if it's Fortran, Pascal, or older C.
2. **Extract Logic:** Isolate the solver loop (e.g., the convergence criteria).
3. **Map Constants:** Identify hardcoded stability constants (log K values).
4. **Report:** Provide a pseudo-code summary of the algorithm before writing any new C++ code.
